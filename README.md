# HexOS

>[!IMPORTANT]
>HexOS is still in very early development and is currently not ready for normal usage.

HexOS is a lightweight easy to use operating made to limit e-waste.

## Dependencies
| Name       | Usage            |
| ---------- | ---------------- |
| nasm       | Assembler        |
| qemu       | Virtualization   |
| bochs      | Debugging        |
| mtools     | Embedding files  |
| dosfstools | File system      |
| gcc        | C/C++ compiler   |
